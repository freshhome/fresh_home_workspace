# دليل هيكلية وتصميم النظام المالي (Financial System Documentation Hub)

مرحباً بك في مركز توثيق وهندسة النظام المالي الخاص بـ **Fresh Home**. يحتوي هذا المجلد على كافة التقارير الفنية، والمخططات المعمارية، وتصميمات الجداول والمحاسبة المعززة بدفتر الأستاذ وقواعد أمان قاعدة البيانات (RLS) ومسارات الحسابات والتسويات التي تم اعتمادها.

---

## فهرس المستندات (Documentation Index)

1. **[تقرير التدقيق والتحليل المالي المبدئي](file:///d:/fresh_home_workspace/docs/financial_system/financial_system_audit_report.md)**
   * **محتواه:** تحليل الوضع المالي الحالي وتحديد الفجوات البرمجية، وتوضيح كيفية احتساب الأرصدة السابقة في كود الفلاتر بدلاً من الخادم، وكيفية تجميد الأسعار التاريخية للحجوزات.
   
2. **[المواصفات الفنية والهندسة المالية للنظام](file:///d:/fresh_home_workspace/docs/financial_system/financial_architecture_specification.md)**
   * **محتواه:** الهندسة الكلية المقترحة (القيود المزدوجة، التسويات، التعديلات الإدارية، إيقاف الفنيين، كشف الحساب المبسط للفني مع إخفاء العمولات التشغيلية).

3. **[تصميم جداول وقاعدة بيانات النظام المالي](file:///d:/fresh_home_workspace/docs/financial_system/financial_database_design.md)**
   * **محتواه:** تصميم الجداول التفصيلي بالخادم، أنواع الحقول والمقيدات الرياضية الصارمة، استراتيجية الفهرسة والتسريع، وسياسات حماية البيانات (Row Level Security - RLS).

4. **[مسارات العمل والتدفق المالي بالتفصيل](file:///d:/fresh_home_workspace/docs/financial_system/financial_workflows.md)**
   * **محتواه:** خطوات وسيناريوهات العمليات المالية (إتمام الطلبات الكاش والإلكترونية، تسويات الحسابات، توليد وتحديث مديونية الفني وسقف الدين، فض النزاعات وفروقات تحصيل النقدية).

5. **[تقرير المراجعة المعمارية - التعديل الأول (Revision v1)](file:///d:/fresh_home_workspace/docs/financial_system/financial_architecture_revision_v1.md)**
   * **محتواه:** توثيق المراجعات والقرارات التقنية المعتمدة في الإصدار الحالي (مثل إلغاء تخزين الرصيد الإجمالي كقيمة مستقلة، وهيكل التعديلات الإدارية المحكم، وقواعد حالات الحساب وعزل العمولات).

6. **[خريطة طريق التنفيذ والتقييم المعماري](file:///d:/fresh_home_workspace/docs/financial_system/implementation_roadmap_financial_system.md)**
   * **محتواه:** خطة التقدم البرمجي للمراحل المستقبلية (الربط البرمجي لتطبيق الفني والمدير، بوابات الدفع، التقارير) مع تحليل نقاط القوة والضعف والمخاطر المعمارية.
